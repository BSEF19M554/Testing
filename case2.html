What is HTML?

Short for HyperText Markup Language, the authoring language used to create documents on the World Wide Web. HTML defines the structure and layout of a Web document by using a variety of tags and attributes.

Uses of HTML:

HTML is used to create web pages, to arrange graphics on a webpage, to link to different pages within a website, and to link to different websites.

Tags and their uses

HTML tag is an HTML code that defines every structure on an HTML page, including the placement of text and images and hypertext links. HTML tags begin with the less-than character (<) and end with greater-than (>). Examples are a, center, title, head, body etc.

Tags Description
Html Describes the browser that this is an HTML document. The html tag represents the root of an HTML document.
HEAD Acts as a container for Head Elements. It includes document-title, scripts, styles, meta information etc.
TITLE Defines the title of the document and is required in HTML document.
BODY Defines the document-body. It contains HTML contents, such as text, hyperlinks, images, tables, lists, etc.
H1-6 Defines HTML headings.
P Defines a paragraph.
BR Inserts a line break.
IMG Defines an image in an HTML page.
Video Specifies video, such as a movie clip or other video streams.
A Used to link from one page to another. It's attribute is the href attribute, which indicates the link's destination.
Center Used to set the align of text into the center.

Attributes and their uses

Attributes provide additional information about an element. All HTML elements can have attributes. Attributes usually come in name/value pairs like: name="value". Examples are align, alt, bgcolor, border, style etc.

Attributes Description
Style Specifies an inline style for an element.
Color Specifies the text color of an element.
Bgcolor Specifies the background color of an element.
Border Specifies the width of the border of an element.
Width Specifies the width of the element, in pixels.
Height Specifies the height of the element, in pixels.
Href Specifies the URL of the page the link goes to.
Colspan Specifies the number of columns a table cell should span.
Wrap Specifies how the text in a text area is to be wrapped when submitted in a form.

Formatting and their uses

HTML also defines special elements for defining text with a special meaning. HTML uses elements like "B" and "I" for formatting output, like bold or italic text. Examples are B, Strong, I, Sub, Sup, Small etc.

Format Description
B Defines bold text, without any extra importance.
I Defines italic text, without any extra importance.
U Underlines text with normal importance.
S Makes text strikethrough, without any extra importance.
Strong Defines strong text, with added semantic "strong" importance.
SUB Defines subscripted text.
SUP Defines superscripted text.
